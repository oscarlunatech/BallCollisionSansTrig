# BallCollisionSansTrig
A simple ball collision simulation that uses linear algebra instead of for the collisions

Two-dimensional elastic collision is a good and visual learning lession traditionally taught in an intro to physics course.
https://en.wikipedia.org/wiki/Elastic_collision#Two-dimensional_collision_with_two_moving_objects
The traditional method uses trigonomitry, while correct calculating trigonometric functions requires complex algorithms that consume significantly more CPU cycles


This assignment was the final to my Computational Science class. It was mostly a fun deep dive into how linear algebra is used in important computations 
such as SVD, Gauss–Seidel method, and Gradient Decent. I thought this would be fun to share since it's a cute visual example.
